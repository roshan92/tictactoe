# tictactoe

There are 3 options to play:-

1. Against human
2. Against newbie computer
3. Against the master aka the unbeatable


The board design (space numbering)

0,0 | 0,1 | 0,2
1,0 | 1,1 | 1,2
2,0 | 2,1 | 2,2


# Get started

1. Install Ruby
2. Clone the repo
3. Navigate to /bin directory, start playing using this command --> ./start.rb
